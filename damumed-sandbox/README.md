# Damumed Sandbox (Demo)

Важно: это не официальный Damumed и не 1:1 клон. Это локальная песочница для демонстрации вашего AI+RPA агента на интерфейсе, похожем по рабочему потоку.

## Что есть

- Открытие первичного приема пациента.
- Гранулярное заполнение полей: жалобы, анамнез, объективные данные, назначения.
- Генерация расписания на 9 рабочих дней (LFK / массаж / психолог) с учетом слотов.
- Назначения со сменой статуса на `Выполнено` + дневниковая запись + timestamp.
- Журнал действий (audit trail).
- Переключаемые профили интерфейса (невропатолог / психолог) с разными дефолтами расписания.
- Профиль психолога: 10 рабочих дней, поддержка конфликтов времени и ветки norm/deviations.

## Запуск

```bash
npm install
npm start
```

Откройте: `http://localhost:4173`

## Зачем это нужно

Чтобы быстро тестировать Chrome Extension / внешний RPA агент без доступа к боевому аккаунту.
